## UI Integration Rules

- Navigation bar must be identical across all pages
- Max content width: 1100px
- Consistent spacing system:
  - small: 8px
  - medium: 16px
  - large: 24px
  - x-large: 32px
- Primary button: filled, high contrast
- Secondary button: outline or subtle background
- Typography:
  - One main heading (h1) per page
  - Section headings use h2/h3 consistently
- Pages must be readable on desktop and mobile
